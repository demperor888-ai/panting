# 河南磐霆速科新材料科技有限公司官网

## 使用方法

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建静态网站
```bash
npm run build
```

构建后的静态文件在 `out` 目录中，可部署到任何静态服务器。

## 网站结构

- 首页 - 轮播Banner、产品展示、公司简介、新闻动态
- 产品中心 - 5个产品系列分类展示
- 新闻资讯 - 公司动态、行业新闻
- 成功案例 - 项目案例展示
- 关于我们 - 公司简介、企业文化、资质荣誉
- 联系我们 - 联系方式、在线留言

## 产品分类

1. 灌浆料系列
2. 建筑砂浆系列
3. 特种结构系列
4. 防水堵漏系列
5. 装配式墙面砂浆系列

## 后台管理

后台管理系统独立部署，位于桌面 `panting-admin` 目录。
